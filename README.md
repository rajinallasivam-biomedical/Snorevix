## 📌 About
Snorevix is an IoT-based smart pillow system that 
detects snoring in real-time, automatically corrects 
head position using air pumps, monitors heart rate, 
and displays live sleep data via Flutter mobile app 
and Firebase cloud.

## ⚙️ Tech Stack
- **Microcontroller:** ESP32-S3
- **Sensors:** MEMS Microphone, MPU6050, Pulse Sensor
- **Cloud:** Firebase Realtime Database
- **Mobile App:** Flutter
- **Communication:** WiFi, I2C, UART
